Fibonacci Search

Fibonacci Search is a comparison-based technique that uses Fibonacci numbers to search an element in a sorted array. Fibonacci search, also known as Fibonacci interpolation search, is a searching algorithm that uses the Fibonacci sequence to divide the search space into smaller intervals. It is an optimization of binary search that works best on uniformly distributed arrays.

How Fibonacci Search Works?

  1. Find the two smallest Fibonacci numbers that are greater than or equal to the length of the search interval. Let these two numbers be F(k) and F(k+1), where F(0) = 0, F(1) = 1, and F(i) = F(i-1) + F(i-2) for i > 1.
  2. Initialize two variables, left and right, to the two ends of the search interval.
  3. Set a third variable, mid, to the smaller endpoint plus the Fibonacci number preceding F(k), scaled to fit within the interval. That is, mid = left + F(k-1)*(right - left)/F(k+1).
  4. Compare the target value with the element at position mid. If they are equal, the search is successful and mid is returned. If the target is smaller than the element at mid, discard the elements from mid+1 to right, set right = mid-1, and recalculate mid using the larger Fibonacci number, i.e., k = k-1. If the target is larger than the element at mid, discard the elements from left to mid-1, set left = mid+1, and recalculate mid using the smaller Fibonacci number, i.e., k = k-2.
  5. Repeat step 4 until either the target is found or left > right, indicating that the target is not in the interval.

Complexity Analysis of Fibonacci Search:

Time Complexity:
  • Best Case: O(1)
  • Worst Case: O(n)
  • Average Case: O(log n)
Auxiliary Space:
O(1) as except the variable to iterate through the list, no other variable is used.

Pseudo Code

function fibonacci_search(array, value):
fib_m2 = 0
fib_m1 = 1
fib = fib_m1 + fib_m2
while (fib < array.length):
fib_m2 = fib_m1
fib_m1 = fib
fib = fib_m1 + fib_m2
// Define two pointers and search the array.
offset = -1
// Offset from the beginning of the array br while (fib > 1):
// Check if fib_m2 is a valid index in the array
i = min(offset + fib_m2, array.length - 1)
if (array[i] < value):
// Move the pointers to the right
fib = fib_m1
fib_m1 = fib_m2
fib_m2 = fib - fib_m1
offset = i
elif (array[i] > value):
// Move the pointers to the left
fib = fib_m2
fib_m1 = fib_m1 - fib_m2
fib_m2 = fib - fib_m1
else:
// Found the value, return its index
return i
// Step 3: Check the last element
if (fib_m1 == 1 and array[offset+1] == value):
return offset+1
// Value not found
return -1
